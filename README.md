# easymsql

## EN:
This script works as an utility or library that extends samp's mysql plugin capabilities.
You will be able to easily create complete schemas using this library, which will improve the strength,
portability of your gamemode and make deployments way easier. 
## ES:
Este script funciona como una utilidad o librería que extiende las capacidades del plugin mysql de samp.
Podrás crear estructuras de datos completas usando esta librería, lo que hará que tu GM sea mas sólida y portable además de  hacer mas facil desplegar el servidor.
